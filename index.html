HTML:

HTML (HyperText Markup Language) is the standard markup language used to structure and design web pages. It defines how text, images, and multimedia content are displayed in a web browser.


Features:

  • Introduction to Audio and Video
  • Vector Graphics
  • Header and Footer

Advantage:

  • HTML helps to build the structure of a website and is a widely used markup language.
  • It is easy to learn.
  • Every browser supports HTML.
